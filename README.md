# 📖 Project Overview

Drug Disease Prediction Using Machine Learning is an intelligent healthcare system that predicts the most appropriate medication for patients based on medical information such as age, gender, blood pressure, cholesterol level, and Sodium-to-Potassium ratio.

The project applies supervised Machine Learning algorithms to analyze healthcare datasets and generate accurate drug recommendations.

It assists healthcare professionals by reducing diagnosis time, improving decision-making, and supporting data-driven medical treatment.

---

# 🎯 Problem Statement

Selecting the right medication depends on several patient-specific medical factors.

Traditional diagnosis is often:

- ⏳ Time-consuming
- ⚠️ Prone to human error
- 📉 Inconsistent
- 🚑 Difficult in remote healthcare

This project solves these challenges by developing an AI-powered drug recommendation system capable of providing intelligent treatment suggestions using Machine Learning.

# ⚡ Installation

## Clone Repository

```bash
git clone https://github.com/medipallysriram/Drug-Disease-Prediction-Using-Machine-Learning.git
```

---

## Go to Project

```bash
cd Drug-Disease-Prediction-Using-Machine-Learning
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

### Linux / macOS

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
python app.py
```

Application will start at:

```
http://127.0.0.1:5000
```
